# This repository contains the comparison between the MitoCore model after alignment to Human1 carried out at the MdOA group (ISAS, Dortmund) 

It contains:
-latest MitoCore model (Cleaned_Mitocore_changed_metabolites_metanetx.chemical.xml)
-Human1 model compared at the time of the project (Human-GEM.xml)
-Coverage of Mitocore by Human1 accessions for alignment (Coverage_Human1)
-Pathway assignment and coverage by each model (KEGG modules)
-Pathway similarity analysis (PCoA)
-General statistics for absolute and relative values in terms of: ec-code, GPR, netabolites, reactions, genes (Statistics)
