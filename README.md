# MitoCore vs Human1 Model Comparison

This repository contains the comparison between the **MitoCore model** (after alignment to **Human1**) carried out by the **MdOA group (ISAS, Dortmund)**.  
It includes data, analyses, and statistics generated during the project.

## Repository Contents

| File / Folder | Description |
|---------------|-------------|
| `Cleaned_Mitocore_changed_metabolites_metanetx.chemical.xml` | Latest MitoCore model used for analysis. |
| `Human-GEM.xml` | Human1 model version used for comparison. |
| `Coverage_Human1` | Coverage of MitoCore metabolites by Human1 accessions for alignment. |
| `KEGG modules` | Pathway assignment and coverage information for each model. |
| `PCoA` | Pathway similarity analysis results using Principal Coordinates Analysis. |
| `Statistics` | General statistics (absolute and relative) for EC codes, GPRs, metabolites, reactions, and genes. |

## Overview

This project aims to:  
- Compare MitoCore and Human1 models after alignment.  
- Assess coverage of MitoCore by Human1 accessions.  
- Analyze pathway assignment and similarity between the models.  
- Generate detailed statistics for metabolites, reactions, genes, EC codes, and GPR associations.
